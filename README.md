# lab 5-6 
Описание проекта: проект, который имитирует работу библиотеки. Это консольное приложение написанное на C++
Требования к проекту: c++17, CMake 3.10+
Примеры использования:
1) Создание пользователя
   Для этого в консольном меню выбирается соответствующий пункт и вводятся соответствующие поля
2) Добавление книги
   Для этого в консольном меню выбирается соответствующий пункт и вводятся соответствующие поля
3) Сохранение данных в файл
  Для этого в консольном меню выбирается соответствующий пункт

Инструкции по сборке
1) написать make CmakeLists.txt
